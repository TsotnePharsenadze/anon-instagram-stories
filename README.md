## Key Features 🌟

- **Anonymous Story Viewing** 🕵️‍♂️  
  View Instagram stories without an account or notifying the user.

- **User Profile Lookup** 🔍  
  Search Instagram profiles and get user information instantly.

## Technology Stack 🛠️

- Next.js 🚀
- TypeScript 🟨
- React ⚛️
- ShadCN 🎭
- React Hook Form 📝
- Prisma 🌌
- TailwindCSS 🎨
